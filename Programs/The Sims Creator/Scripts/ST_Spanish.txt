The right side of the panel has the color swatch and a color picker. The color swatch shows the current foreground color, background color, and a few preset colors. The foreground color will be the color used with the Color Paintbrush tool and the background color will be the color used with the Eraser tool. You can choose a preset color by left clicking, or store the foreground color by right clicking. Clicking within the color picker changes the current foreground color.|La parte derecha del panel contiene la muestra del color y un selector de color. La muestra de color indica el color que se halla en primer plano, el color de fondo y algunos colores predeterminados. El color en primer plano es el color que se utilizará con la herramienta Pincel de color y el color de fondo es el color que se utilizará con la herramienta Goma de borrar. Puedes elegir un color predeterminado haciendo clic izquierdo o guardar el color de primer plano haciendo clic derecho. Si haces clic dentro del selector de color cambiarás el color de primer plano.
